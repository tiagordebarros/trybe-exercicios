### HTML - Primeiras tags

#### Para fixar

```
index.html
```

Copiar

```html
<!DOCTYPE html>
<html lang="pt-br">
  <head>
    <meta charset="UTF-8">
    <title>HTML</title>
  </head>
  <body>
    <h1>Exercícios</h1>
  </body>
</html>
```

**1** - Crie uma lista das pessoas que você já fez amizade na Trybe.

**2** - Adicione uma imagem que venha de uma URL externa.

**3** - Adicione uma imagem que está no seu computador, local.
